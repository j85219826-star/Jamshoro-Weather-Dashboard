# Jamshoro Weather Dashboard

A simple 3‑page weather dashboard built with **HTML, CSS, and JavaScript**.  
This project displays **hardcoded weather data for Jamshoro, Sindh** (today’s snapshot) without requiring an API key.

## 📂 Project Structure
- `index.html` → Main dashboard showing today’s weather summary.
- `style.css` → Stylesheet for consistent design across all pages.
- `script.js` → Injects Jamshoro’s weather data into the dashboard.
- `details.html` → Extra page with sunrise, sunset, visibility, and AQI details.

## 🚀 How to Run
1. Download or clone this repository.  
2. Open `index.html` in your browser.  
3. Navigate between pages using the links provided.

## 🌐 GitHub Pages
You can host this project live using **GitHub Pages**:
- Go to **Settings → Pages** in your repository.
- Select branch `main` and root folder.
- GitHub will generate a link like:  
  `https://yourusername.github.io/jamshoro-weather-dashboard/`

## 📖 Notes
- This version uses **hardcoded weather data** for Jamshoro.  
- If you want live updates, you can connect it to the **OpenWeatherMap API** by adding your API key in `script.js`.
